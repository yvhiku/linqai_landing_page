# LINQ AI Landing Page

Simple landing page project built with React + Vite.

## Requirements

- [Node.js](https://nodejs.org/) 18+ (recommended: latest LTS)
- npm (comes with Node.js)

## Getting Started

1. Clone the repository:

```bash
git clone <your-repo-url>
cd linqai
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the app in your browser:

- [http://localhost:5173](http://localhost:5173)

## Available Scripts

- `npm run dev` - Starts Vite dev server
- `npm run build` - Creates production build in `dist/`
- `npm run preview` - Serves the production build locally

## Production Build

Build and preview locally:

```bash
npm run build
npm run preview
```

## Project Structure

- `src/app/` - Main app and page sections/components
- `src/styles/` - Global styles and theme files
- `index.html` - Vite HTML entry
- `src/main.tsx` - React app bootstrap

## Notes

- If `pnpm` is not installed, use `npm` commands (as shown above).
- If port `5173` is already in use, Vite may choose another port automatically.
